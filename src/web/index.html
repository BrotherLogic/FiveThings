FiveThings

Essence

FiveThings is a simple daily task manager. In essence you define five things which you would like to achieve on a single a machine for a single day. You then tick them off when they are complete. When all the tasks are complete you are done and can enjoy the rest of the day.

Running

FiveThings runs on top of JNLP. Click the link below to run and install the program on your machine. You can put it where you wish using the standard approach but it should appear in your start menu or applications folder. You will need to install java if you have not done so already. It will update itself automatically once installed.
